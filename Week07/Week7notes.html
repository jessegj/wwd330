    Immediately Invoked Function Expressions (IIFE) – an anonymous function that is invoked as soon as it is
    defined. This is a good way to use temporarily variables. They will only be used when the function is
    called and then they will disappear. This also helps create code blocks that are able to run independently
    of each other.

    Callbacks are functions passed to other function and then invoked inside the function they are passed
    to. This helps the code to be processed out of order or asynchronously. You have to be careful with
    Callbacks so we do not create spaghetti code.

    Promise represents a future result of an asynchronous operation. A promise runs through a cycle. After
    it is created it goes to pending while it takes place. Then to settled promise can have a resolved or
    rejected result. It is created using a constructor function.
